# Modified Ask Set

A quick and ugly modification to the Vanilla Ask Set effects for Valheim

## Installation (manual)

Grab the ModifiedAskSet.dll file and place it in the Bepinex plugins folder


## Features

Treat yourself to an overpowered Ask Set
